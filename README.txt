Snake-Online

Snake-Online - это многопользовательская версия классической игры "змейка", где игроки играют онлайн и соревнуются между собой, пытаясь стать самой длинной змеей.

Инструкции по установке:
1.Перейти по ссылке: https://disk.yandex.ru/d/qGm-ZiqtOX3IKA
2.Скачать файл Snake-Online

Для запуска игры требуется:
1. Установить файл Snake-Online из него открыть файл dist, в котором будет 2 файла формата .exe: Server.exe и SnakeClient.exe
2. С помощью файла settings.txt настроить нужную конфигурацию игры
3. Запустить приложение Server.exe.
4. Запустить приложение SnakeClient.exe

Инструкции по игре:
Управление змейкой происходит клавишами: WASD.
Красный квадрат - яблоко, которое при поглащении увеличивает вашу змейку.
Серый квадрат - камень, при столкновении с ним змейка умирает.
Фиолетовый квадрат - телепоритрует змейку на другой случайный телепорт.
Избегайте столкновений вашей змейки с хвостом других змеек.
Цель набрать как можно больше очков, чтобы вырваться на 1 место в таблице лидеров на сервере.
